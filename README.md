

# Lumina AI - Video Editing Supercharged

Welcome to Lumina AI, a modern and responsive landing page designed to showcase an AI-powered video editing platform. This application highlights key features, user testimonials, and provides a clear call to action, all wrapped in an engaging and animated user interface.

## Features

*   **Animated Background:** A subtle, dynamic gradient background enhances the visual appeal.
*   **Responsive Navbar:** A sticky navigation bar with branding and call-to-action buttons.
*   **Hero Section:** An impactful introduction with a compelling headline, description, and interactive buttons, featuring glowing orb animations.
*   **Social Proof:** A marquee-style scrolling section showcasing logos of industry leaders, building trust.
*   **Bento Grid Features:** A visually appealing grid layout highlighting core AI-powered editing capabilities (Auto-Captions, Magic Cut, Neural Color Grade) with hover animations.
*   **How It Works:** A step-by-step guide explaining the user workflow, with animated cards and connecting lines.
*   **Testimonials:** A section displaying positive feedback from users with star ratings and animated cards.
*   **Newsletter Signup:** A form for users to subscribe to updates, featuring form validation and toast notifications.
*   **Call to Action:** A prominent section encouraging users to start editing, with a shimmering button effect.
*   **Footer:** A clean footer with copyright information and a link to Dyad.
*   **Global Styling:** Utilizes Tailwind CSS for a consistent and modern design.
*   **Routing:** Basic routing with React Router for the main page and a 404 Not Found page.

## Tech Stack

*   **React:** A JavaScript library for building user interfaces.
*   **TypeScript:** A typed superset of JavaScript that compiles to plain JavaScript.
*   **Tailwind CSS:** A utility-first CSS framework for rapidly building custom designs.
*   **Framer Motion:** A production-ready motion library for React.
*   **React Router:** Declarative routing for React.
*   **shadcn/ui:** Reusable components built with Radix UI and Tailwind CSS.
*   **react-hook-form:** For flexible and extensible forms with easy validation.
*   **zod:** A TypeScript-first schema declaration and validation library.
*   **sonner:** A modern toast component for React.
*   **lucide-react:** A collection of beautiful & consistent icons.
*   **Vite:** A fast build tool for modern web projects.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

*   Node.js (v18 or higher recommended)
*   npm or Yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd <repository-directory>
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

    The application will be available at `http://localhost:8080`.

## Usage

This project is a landing page. Simply open it in your browser and scroll through the sections to experience the design and features.
